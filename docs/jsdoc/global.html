(constant) mesa_montada :string

HTML da mesa de jogo, composta por 4 linhas com 3 cartas cada (direita, centro, esquerda). Cada carta pode ser interativa (arrastar e soltar) ou coringa (central). Utiliza classes para controle visual e identificadores únicos para facilitar a lógica do jogo. IDs seguem o padrão: - `direita-{n}`, `centro-{n}`, `esquerda-{n}` (n variando de 1 a 4) Este conteúdo deve ser inserido dinamicamente no DOM antes da animação ou lógica de interação.
